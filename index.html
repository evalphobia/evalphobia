Go

Wizard

Wizard is the database connection library, which supports sharding and multiple instances.
xorm is used in wizard. (thanks lunny)

aws-sdk-go-wrapper

aws-sdk-go-wrapper is wrapper library of aws-sdk-go.
This enables to use AWS SDK in golang with easy syntax.

Supported services below,

  • DynamoDB
  • Kinesis
  • S3
  • SNS
  • SQS

google-api-go-wrapper

google-api-go-wrapper is wrapper library for Google services.

Supported services below,

  • Google Analytics
  • BigQuery

eurekache

eurekache is cache library, which supports multiple fallbacking cache chains.

go-paypal-classic

go-paypal-classic is PayPal library for Classic API.

go-iap

go-iap is forked from dogenzaka/go-iap.
This version supports old version IAP receipts(iOS6 style receipt).

go-yml2sql

go-yml2sql is library to create SQL statement from YAML file.
This supports CLI and import code.

kinesis-tail

kinesis-tail is tailing tool from Amazon Kinesis Stream.

Logrus hooks

These are plugins to sends log data to external services for logrus.

  • logrus_appneta
    • TraceView hook
  • logrus_kinesis
    • AWS Kinesis hook
  • logrus_sentry
    • Sentry hook
  • logrus_fluent
    • fluentd hook
